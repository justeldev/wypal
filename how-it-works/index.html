Simple Process

How Wypal Works

A clear, privacy-first pipeline built for speed and trust.

  1. 1
    Paste or Upload

    Paste AI output into the editor or drag in .txt, .md, or .docx files. Batch uploads supported.

  2. 2
    Toggle Cleaning Options

    Enable removal of markdown (#, **, [ ]( )), typography (— …), and AI patterns. Changes preview instantly.

  3. 3
    Diff for Confidence

    Switch to Diff View to compare Before/After side-by-side with precise highlights of edits.

  4. 4
    Export & Share

    Copy to clipboard or download as .txt. In batch mode, download each cleaned file instantly.

What gets cleaned

Markdown

Headers (# ##), bold (** **), italic (* *), lists (- 1.), links ([ ]( )), code (` `, ``` ```), quotes (> ), tables (| |).

Typography

Smart quotes (" " ' '), em/en dashes (— –), ellipses (…), bullets (• ◦ ▪).

AI Patterns

Formulaic openings ("Here's a..."), transitions (Furthermore, Moreover), hedging (It's worth noting...), parenthetical noise.

Why the Diff View matters

See exactly what changed — nothing hidden.

Validate replacements (e.g., em dash → comma) before finalizing.

Combine with Undo/Redo to audit steps safely.
